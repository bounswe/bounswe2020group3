# CmpE 352 & 451 2020 Group 3

# Description

Welcome to our github repository for Bogazici University's Fundamentals of Software Engineering courses, CMPE352 and CMPE451. We are working on an Academic Collaboration Platform project as group 3. You can view our [Wiki Page](https://github.com/bounswe/bounswe2020group3/wiki) for details.

# Project

Our project aims to provide an colloborative platform for academicians where they can find colloborators, widen their networks and provide an environment to create interdisciplinary studies.
